# Subnautica 2 Mod Menu

A feature-rich internal DLL mod menu / game trainer for Subnautica 2 built with C++ and ImGui.

## Features

### Player Cheats
- **God Mode** - Infinite health, prevents all damage
- **Infinite Oxygen** - Never run out of oxygen underwater
- **Speed Hack** - Adjustable movement speed multiplier (1x-10x)
- **No Clip** - Fly through walls and objects

### Tool Cheats
- **Infinite Battery** - Tools never run out of energy

### ESP (Extra Sensory Perception)
- **Material ESP** - Highlight materials/resources
- **Player ESP** - See other players with distance
- **Creature ESP** - See creatures with distance
- **Item ESP** - See items/pickups with distance

## Building

### Prerequisites
- Visual Studio 2022 or later
- CMake 3.20 or later
- Windows 10/11 SDK

### Setup

1. **Clone ImGui** (vendor dependency):
```powershell
cd vendor
git clone https://github.com/ocornut/imgui.git
cd imgui
git checkout docking  # or latest stable
```

2. **Configure CMake**:
```powershell
mkdir build
cd build
cmake .. -G "Visual Studio 17 2022" -A x64
```

3. **Build**:
```powershell
cmake --build build --config Release --target Subnautica2ModMenu 2>&1 | Select-String -Pattern 'error','fatal'
```

The DLL will be output to `build/bin/Release/Subnautica2ModMenu.dll`.

## Usage

1. Inject the DLL into Subnautica2.exe using your preferred injector
2. Press **INSERT** to toggle the mod menu
3. Enable/disable features as desired

## SDK Integration

This project uses the SDK generated by [Dumper-7](https://github.com/Encryqed/Dumper-7) from the Subnautica 2 game dump.

Key classes used:
- `ASN2PlayerCharacter` - Local player character
- `ABP_SN2PlayerController` - Player controller
- `UUWESurvivalSetComponent` - Oxygen/food/water stats
- `UUWEMechanicalSetComponent` - Battery/energy stats
- `UUWEMovementSetComponent` - Movement speed stats
- `UUWEHealthSetComponent` - Health stats
